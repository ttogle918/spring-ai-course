# 이미지 임베딩

목적

- 이미지(또는 이미지에서 추출한 특징)를 임베딩으로 변환해 시각적 유사도 검색이나 이미지-텍스트 결합 검색에 사용합니다.

관련 파일

- 템플릿: `src/main/resources/templates/image-embedding.html`
- (이미지 임베딩 호출은 프로젝트의 기본 임베딩 API를 사용하도록 구성되어 있음)

핵심 흐름

1. 클라이언트에서 이미지를 업로드한다.
2. 서버는 이미지 임베딩 엔드포인트(또는 모델)를 호출해 벡터를 생성한다.
3. 생성된 벡터를 `VectorStore`에 저장하거나 즉시 검색에 사용한다.

확인 포인트

- 업로드된 파일이 모델에 전달되는지 확인
- 생성된 벡터의 차원과 값 범위를 확인

팁

- 이미지 전처리(크기 조정, 정규화)가 결과에 영향을 줄 수 있으므로 일관된 전처리 파이프라인을 유지하세요.
